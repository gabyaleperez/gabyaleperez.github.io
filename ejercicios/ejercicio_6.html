La única forma de saber si un texto es un enlace es colocando el cursor sobre el texto y observar si el cursor cambia de forma a una mano, lo que indica que el texto es un enlace y se puede hacer clic sobre él para acceder a una página web o recurso en línea.

En este ejercicio, se presenta una lista de palabras y se pide buscar el enlace colocando el cursor sobre cada una de ellas y observando si el cursor cambia de forma a una mano. Una vez que se ha identificado la palabra que es el enlace, se pide al usuario que haga clic sobre ella para acceder al recurso en línea correspondiente.
